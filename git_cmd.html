1. Git status
        - trạng thái hiện tại của répo mà mình đang dùng
        2. git branch 
            - tạo một nhánh mới từ nhánh hiện hành
        3. git checkout 
            - chuyển trạng thái qua một nhánh đã tạo
        4. git commit -m ""
        - tạo một cam kết khi chúng xác một file hoặc nhiều file thay đổi
        5. git push origin 
        - đẩy những thay đổi mới nhất mà chúng ta đã cam kết lên origin
    6. git add
